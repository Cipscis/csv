CSV 1.0.3 - Updated build system and added build-and-deploy action

Back to changelog

This patch updated the build system to use a more up to date SASS compiler, and added a GitHub Action to build and deploy the package to GitHub Pages on push. As a result, the compiled CSS and bundled JS files are now ignored.

Breaking changes

  • None

Non-breaking changes

  • Use sass to compile CSS, instead of the deprecated node-sass compiler used by gulp-sass by default.
  • Added a GitHub Action to build the project on push to the main branch, then deploy the built project to the gh-pages branch.
